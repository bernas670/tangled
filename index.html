How To Play

Solve two interlocking word puzzles on a 5×5 grid.

  • Fill each row and column with valid 5-letter words.
  • Every letter must work for both its row AND column word.
  • Toggle between Row and Column mode to submit words.

Tile Colors

A

Green — Correct position.

B

Yellow — In both the row and column word, but wrong spot.

C

Cyan — In the row word only, but wrong spot.

D

Pink — In the column word only, but wrong spot.

E

Gray — Not in either word.

Grid Layout

The panels around the grid track your knowledge:

  • Left/Right panels — Misplaced and absent letters for each row.
  • Top/Bottom panels — Misplaced and absent letters for each column.

Controls

Keyboard

  • Arrow keys — Move cursor.
  • Space — Toggle Row/Column mode.
  • Enter — Submit current line.
  • Backspace — Delete letter.
  • Esc — Toggle this modal.

Mouse

  • Click — Select a cell.
  • Double-click — Select cell and toggle mode.

Controls

  • Tap — Select a cell.
  • Double-tap — Select cell and toggle mode.
  • Use the on-screen keyboard to type.
